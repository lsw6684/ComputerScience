## Foundation
 - 보안의 3대 요소
    - **기밀성 Confidentiality** <br />
        인가(Authorization)된 사용자만 정보 자산에 접근할 수 있는 것을 말합니다. `암호, 패스워드`
    - **무결성 Integrity** <br />
        적절한 권한을 가진 사용자에 의해 인가된 방법으로만 정보를 변경할 수 있도록 합는 것을 말합니다. `개인정보 수정`
    - **가용성 Availability** <br />
        정보 자산에 대해 적절한 시간에 접근 가능한 것을 의미합니다. `24시 편의점`
